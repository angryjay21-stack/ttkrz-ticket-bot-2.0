Railway:
Start command is npm start

Required Railway variables:
TOKEN
GUILD_ID
TICKET_CHANNEL_ID
STAFF_ROLE_ID

Optional:
TICKET_CATEGORY_ID

The bot posts the ticket panel automatically when it starts.
Users open tickets from the dropdown.
Only the user and staff role can see the ticket channel.
Staff can claim tickets.
Staff can close tickets.
